# macpan2 Calibration Working Group

10am Wed March 4, CSF 4333

10am Wed March 11, CSF 4333

Please email ahurford@mun.ca if you would like to participate remotely

Topics will be:
- Model calibration with macpan2 (using TMB)
- Bayesian parameter estimation
- Estimating the time-varying transmission rate

Resources:
- [Piecewise transmission rate and splines demo repo](https://github.com/wzmli/macpan2_calibration_lab) by Mike Li
- [Past macpan2 workshops](https://canmod.github.io/macpan-workshop/)
- [macpan2 documentation](https://canmod.github.io/macpan2/)
- [Scarlet fever](https://github.com/canmod/macpan2/blob/main/vignettes/real_data.Rmd)

Goal: Master the materials above and then work on an extension of this framework of your choosing such as:
- Applying the methodology for another disease
- Investigating if we can do spatial structure
- Bayesian parameter estimation (prior uncertainty)
- Extended importation modelling

The idea is to learn these methods, work on a problem of your choosing, and report back to the group about the results of our analysis.

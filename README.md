## Topics
- Model calibration with macpan2 (using TMB)
- Bayesian parameter estimation
- Estimating the time-varying transmission rate

## Goal
Master the materials above and then work on an extension of this framework of your choosing such as:
- Applying the methodology for another disease
- Investigating if we can do spatial structure
- Bayesian parameter estimation (prior uncertainty)
- Extended importation modelling

The idea is to learn these methods, work on a problem of your choosing, and report back to the group about the results of our analysis.

# macpan2 Calibration Working Group

10am Wed March 4, CSF 4333. Please email ahurford@mun.ca if you would like to participate remotely
10am Wed March 11, CSF 4333

Topics will be:
- Model calibration with macpan2 (using TMB)
- Bayesian parameter estimation
- Estimating the time-varying transmission rate

Resources:
- [Scarlet fever](https://github.com/canmod/macpan2/blob/main/vignettes/real_data.Rmd)

And then we will work on an extension of this framework (of our choosing) such as:
Applying the methodology for another disease
Investigating if we can do spatial structure
Bayesian parameter estimation (prior uncertainty) (Francis, did you want to try this one for the Scarlett fever data?)

The idea is to co-work on these problems and report back to the group about the results of our analysis. Josh and I are going to work on applying these methods to his HPAI mortality data.
